# sasaki-agent

Агент поиска вакансий. Пайплайн: `src/collect.mjs` по расписанию (launchd/cron,
каждые 6 ч) собирает вакансии из источников, фильтрует по `config.json`
(ключевые слова/стоп-слова), шлёт новое в Telegram, дедуплицирует через
`data/seen.json`. Настройка с нуля: `npm run setup`.

Личное пользователя — `config.json`, `.env`, `profile.md`, `master-cv.md`,
`data/` — в гит не попадает (см. .gitignore). Профиль для оценки релевантности
и переписки CV — в `profile.md`.

## Конвенции

- Zero-dependency Node (>= 21, используется `Map.groupBy`), ES-модули `.mjs`;
  исключение — Telethon-утилита `src/telegram/tg_fetch.py` (Python через `uv`):
  нужна только для разворачивания папок-инвайтов addlist, НЕ для чтения каналов
  (публичные каналы читаются веб-лентой t.me/s/ в `src/sources/telegram.mjs`)
- Новый источник = файл в `src/sources/`, возвращает массив
  `{ source, id, title, company, url, salary, location, published }`
  (+ `prefiltered: true`, если источник фильтрует сам) и подключается
  в `src/collect.mjs`
- Секреты только в `.env`; правила поиска — в `config.json`
  (шаблон — `config.example.json`)

## Известные тупики (не тратить время повторно)

- Сбер (rabota.sber.ru): SSR-only поиск за бот-протекшеном F5, публичный
  `publications` отдаёт 10 последних и игнорирует параметры → только hh.ru
- Контур: бот-протекшен с редирект-петлёй на весь kontur.ru → только hh.ru
- Озон: JS-челлендж; их вакансии имеют `hhId` и целиком дублируются на hh.ru
- LinkedIn job-постинги — можно (гостевой jobs-guest эндпоинт, без логина,
  см. sources.local/linkedin.mjs); а вот посты в ЛЕНТЕ через аккаунт в крон
  нельзя (бан) — только вручную через браузер
- geekjob.ru: SPA, все /json/* роуты и даже JS-бандлы отдают заглушку-оболочку
  не-браузерам; сайт нестабилен (статика падает и в браузере). Вместо адаптера
  подключён их телеграм-канал @geekjobs (config → telegram.channels)
- Самокат (samokat.tech): JS-челлендж servicepipe — как у Озона, не обойти
- ЦИАН (team.cian.ru): статичный лендинг, все подстраницы (/vacancies и пр.)
  отдают 403 — вакансии публикуют только на hh.ru

## Планы

- Скилл `tailor-cv`: master-cv.md + вакансия → адаптированное CV
- LLM-скоринг релевантности (`claude -p`) поверх фильтра ключевых слов
- Публичная версия (в работе). Онбординг без вопросов: `npm run setup` ставит
  автозапуск и открывает пульт, критерии выбираются на первом экране пульта
  (флаг `onboarded` в config.json)
