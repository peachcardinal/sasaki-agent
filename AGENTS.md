# AGENTS.md

Инструкции для любого ИИ-ассистента (Claude Code, Cursor, ChatGPT/Codex и т.п.),
который помогает пользователю с этим проектом. Пользователь-человек читает
[README.md](README.md); этот файл — как агенту здесь работать.

## Что это

`sasaki-agent` — личный агент поиска работы. Пайплайн `src/collect.mjs` по
расписанию (крон/launchd, каждые 6 ч) собирает вакансии из источников,
фильтрует по `config.json` (ключевые/стоп-слова, грейд, формат), шлёт новое
в Telegram/файлы/CSV/вебхук и дедуплицирует через `data/seen.json`.
Есть и веб-пульт (`npm run ui`, `src/server.mjs` + `src/ui/index.html`) — тот же
`config.json` и `data/`, но с лентой, фильтрами и трекером откликов.

## Конвенции

- **Zero-dependency Node** (>= 21, используется `Map.groupBy`), ES-модули `.mjs`.
  Не добавляй npm-зависимости без явной просьбы. Исключение — Python-утилита
  Telethon (`src/telegram/tg_fetch.py`, через `uv`) для разворачивания
  папок-инвайтов; чтение публичных каналов идёт веб-лентой без неё.
- **Секреты — только в `.env`**, правила поиска — в `config.json`
  (шаблоны: `.env.example`, `config.example.json`). Никогда не коммить
  `config.json`, `.env`, `profile.md`, `master-cv.md`, `data/`, `sources.local/`
  — они в `.gitignore` и содержат личное.
- **Новый источник** = файл-адаптер, возвращающий массив
  `{ source, id, title, company, url, salary, location, published }`
  (+ `prefiltered: true`, если источник фильтрует сам). Личные адаптеры
  пользователя живут в `sources.local/` (не в гите), готовые примеры —
  в `examples/sources/`. Автодискавери подхватывает оба каталога.

## Как добавить источник

Полное руководство (где искать данные, чек-лист, когда сдаться) —
в [.claude/skills/add-source/SKILL.md](.claude/skills/add-source/SKILL.md).
Кратко: напиши адаптер в `sources.local/<имя>.mjs` по интерфейсу выше, включи
его в `config.json → sources`, прогони `node src/collect.mjs` и проверь, что
источник появился в логе и ссылки открываются. Семь рабочих примеров разбора
(RSS, скрытый JSON-API, SSR-HTML, AJAX) — в `examples/sources/`.

## Известные тупики (не трать на них время)

Сайты за бот-протекшеном (F5/DDoS-Guard, JS-челлендж, SSR-only выдача только
браузерам) обходить не нужно — почти все такие компании дублируют вакансии на
hh.ru (источник `hh` с бесплатным токеном). Конкретный список уже проверенных
тупиков (Сбер, Контур, Озон, geekjob, Самокат, ЦИАН и др.) — в
[CLAUDE.md](CLAUDE.md), раздел «Известные тупики».

## Проверка изменений

- Синтаксис: `node --check <файл>`.
- Пайплайн целиком: `node src/collect.mjs` (сходит в сеть, покажет лог по
  источникам). Отдельный источник проще проверить временным скриптом, вызвав
  его `fetch(cfg)` и распечатав первые элементы.
- Пульт: `npm run ui`, затем `http://127.0.0.1:4321` — слушает только localhost.
